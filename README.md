## 一、研究目标

针对肠镜图像中的息肉分割任务，复现经典 U-Net，并加入注意力机制与混合损失函数，改善小息肉、边界模糊及背景干扰条件下的分割效果。

数据集建议采用 Kvasir-SEG。评价指标使用 Dice、IoU、Precision、Recall，并配合典型预测图进行定性分析。

## 二、创新点设计

创新不宜表述为“提出全新网络”，而应表述为面向息肉分割问题的组合式改进方案：

1. 在 U-Net 跳跃连接中引入 Attention Gate，使解码阶段更关注疑似息肉区域，抑制肠壁纹理、反光和器械等背景干扰。

2. 使用 BCE + Dice 混合损失。BCE 保证像素级分类稳定，Dice Loss 缓解息肉区域相对较小导致的前景背景类别不平衡。

3. 可选的轻量补充：在训练阶段加入随机旋转、翻转和颜色扰动，增强模型对拍摄角度与光照变化的鲁棒性。

最终方法可命名为：Attention-HybridLoss U-Net。

与已有工作的区别：不单独复现 Attention U-Net，也不只替换损失函数；而是将“区域注意力 + 类别不平衡优化 + 医学图像增强”组合为一套面向息肉图像的轻量方案，并通过消融实验验证每部分作用。

## 三、实验方案

1. 数据准备  
下载 Kvasir-SEG，统一调整图像尺寸至 256×256。按 8:1:1 划分训练、验证、测试集，或采用固定随机种子划分。

2. 基线实验  
训练原始 U-Net，记录训练曲线、Dice、IoU、Precision、Recall，并保存若干预测图。

3. 消融实验  
分别训练以下四组：

| 编号 | 模型/设置 | 目的 |
|---|---|---|
| A | U-Net + BCE | 基线 |
| B | U-Net + BCE + Dice | 验证混合损失 |
| C | Attention U-Net + BCE | 验证注意力机制 |
| D | Attention U-Net + BCE + Dice | 最终方案 |

4. 参数调试  
至少记录两类调参过程：学习率，如 `1e-3、1e-4、3e-4`；混合损失权重，如 `BCE:Dice = 1:1、1:2、2:1`。不需要全排列穷举，选择少量有代表性的设置即可。

5. 结果讨论  
比较四组指标与可视化结果，重点观察小目标、边界模糊、强反光和复杂背景样本。分析注意力模块和混合损失分别改善了什么，以及仍然失败的案例。
